🚀 AI Resume Parser & Recruitment Platform

An AI-powered recruitment platform that streamlines hiring by analyzing resumes, generating ATS scores, and intelligently ranking candidates for recruiters.

✨ Features

👤 Candidate Portal

* Resume analysis with skill extraction
* ATS compatibility scoring
* Skill gap detection
* Domain prediction

🏢 Recruiter Dashboard

* Bulk resume parsing
* AI-based candidate ranking
* Resume-to-job description matching
* Candidate comparison & shortlisting

🛠️ Tech Stack

Frontend: React.js, HTML, CSS
Backend: Node.js / Express.js
AI/ML: NLP, Resume Parsing Models
Database: MySQL / PostgreSQL

📊 Scoring System

The platform uses a Hybrid Scoring Model:

* Theoretical Score: Resume structure & keyword analysis
* AI Score: Semantic skill and experience evaluation
* Final Score: Combined score for accurate candidate assessment

🎯 Objective

To make recruitment faster, smarter, and more efficient through AI-driven resume screening and candidate evaluation.

🚀 Future Improvements

* Interview question generation
* Resume improvement suggestions
* Recruiter analytics dashboard
* AI job recommendation system

Built to bridge the gap between talented candidates and smarter hiring decisions.
